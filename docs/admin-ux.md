# Admin UX

Die Admin UI folgt einer SaaS-Informationsarchitektur:

- Dashboard
- Seiten
- Collections
- Medien
- Navigation
- Design
- SEO
- Formulare
- Integrationen
- Rechtliches
- Einstellungen
- Benutzer

Der Page Editor besteht aus Section-Liste links, Live-Preview in der Mitte und Section Library beziehungsweise Field Drawer rechts.
