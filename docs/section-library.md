# Section Library

Jede Section enthält:

- `type`
- `label`
- `description`
- `category`
- `icon`
- `tags`
- `schema`
- `defaultData`
- `defaultDesign`
- `defaultAnimation`
- `adminFields`
- Renderer

Die Library ist nicht seitengebunden. Sections können grundsätzlich auf jeder CMS-Seite hinzugefügt werden.
